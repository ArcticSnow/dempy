# README
S. Filhol, March 2026

This is a fresh start of `dempy` that had become to combersome to track. `dempy` intent to bring together a collection of tools to work with Digital Elevation Models. `dempy` started some years ago under Python 2.7 and could use some refreshing and cleanup.

The intent here is to convert to a cleaner toolbox, integrated around xarray dataset and prevent using raster files completely. This is particularly interesting to work from memory when working on small projects.
