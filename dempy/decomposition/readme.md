To install diamond square:

```sh
# create a clean virtual environment
conda create -n diamond
conda activate diamond
conda install matplotlib

# Bring your terminal console within the /dempy/dempy/decomposition folder
cd ~/dempy/dempy/decomposition

# run the setup.py install file. 
python setup.py install

# To test if the install worked you can run the script
python test_diamondSquare.py
```
